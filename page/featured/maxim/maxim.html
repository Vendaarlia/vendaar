Color Tokens

All colors in this system are defined as tokens. Do not use hex values directly in code — always reference tokens.

Brand Palette 4 tokens
color.background
#FDF5EB
Main page background — warm cream
color.primer
#8B3734
Primary brand color — navbar, buttons, accents
color.dark
#000000
Main text — headings and body text
color.light
#FFFFFF
White — card backgrounds, text on primary
CSS Custom Properties globals.css
/* globals.css — declare once, use everywhere */ :root { --color-background: #FDF5EB; --color-primer: #8B3734; --color-dark: #000000; --color-light: #FFFFFF; } /* Tailwind arbitrary values */ /* bg-[#FDF5EB] → background */ /* bg-[#8B3734] → primer */ /* text-[#000] → dark */ /* text-[#FDF5EB] → light on primer */

Do & Don't

✓ Do
Use btn-primer with color #8B3734, border-radius 15px, font Roboto capitalize.
✕ Don't
Don't create buttons with different colors or radius from the established design tokens.
✓ Do

Hair Transplant
Specialists

Use Roboto SemiBold 75px for H1, always with fontVariationSettings: "'wdth' 100".
✕ Don't

Hair Transplant
Specialists

Don't change font family. Georgia, Arial, or other fonts don't match MAXIM's brand identity.
✓ Do
Image grid uses 11px gap with border-radius 27–36px.
✕ Don't
Don't use gaps that are too narrow or border-radius that are too small for images.

Contribution Guidelines

Any changes to the design system must go through review from at least 1 designer and 1 developer. Update tokens in Tokens Studio first, then update this file, then bump version using semantic versioning (MAJOR.MINOR.PATCH).
